
# Student Details Project

A full-stack web app to manage student records — built with **Next.js**, **Django REST Framework**, and **PostgreSQL**, containerized with **Docker**.

Built by **Me** (Frontend) & **[@srinudata](https://github.com/srinudata)** (Backend)

---

## Tech Stack

| Layer | Technology | Port |
|-------|-----------|------|
| Frontend | Next.js | 3000 |
| Backend | Django + DRF | 8000 |
| Database | PostgreSQL 16 | 5432 |

---

## Getting Started

### Prerequisites
- [Docker Desktop](https://www.docker.com/products/docker-desktop/) installed and running
- Git installed

### Run the project

```bash
git clone https://github.com/youruser/student-details-project
cd ./student-details-project
docker compose up --build
```

### Access the app

| Service | URL |
|---------|-----|
| Frontend | http://localhost:3000 |
| Backend API | http://localhost:8000 |
| Database | http://localhost:5432 |
